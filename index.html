Privacy Policy

SanMaoWHV (三猫插件 · NZ WHV 抢签助手)

Last updated: 2026-05-15

1. Data Collection

This extension does NOT collect, transmit, or share any user data with external servers. All personal information (name, passport details, address, health and character declarations, etc.) entered by the user is stored exclusively in the browser's local storage (chrome.storage.local) on the user's own device.

2. Data Storage

  • Form data is saved locally using Chrome's Storage API.
  • Data is never synced to the cloud, never uploaded to any server, and never shared with third parties.
  • Users can delete all stored data at any time by clearing the extension's storage or uninstalling the extension.

3. Permissions Usage

  • storage — Save user's form data locally for auto-fill.
  • activeTab — Access the current tab to fill form fields when the user clicks "Fill now".
  • scripting — Inject content scripts into Immigration NZ pages to perform auto-fill.
  • tabs — Open new tabs for multi-session workflow.
  • Host permissions (immigration.govt.nz) — Run content scripts only on the target visa application site.

4. Third-Party Services

This extension does not integrate with any analytics, advertising, or tracking services.

5. Data Security

All data remains on the user's local machine. The extension does not make any outbound network requests to transmit user data. License verification is performed entirely client-side using cryptographic signatures.

6. Children's Privacy

This extension is not directed at children under 13. We do not knowingly collect personal information from children.

7. Changes to This Policy

We may update this privacy policy from time to time. Any changes will be reflected on this page with an updated revision date.
